Anthropic Financial Agents
完整開發指南

從零開始，把 Claude Managed Agents（CMA）開發、整合、部署到銀行的 AWS 環境，並讓它安全地上線營運 — 一本寫給工程師的實戰手冊。

這本書怎麼讀 第 1–3 章建立地基；第 4–6 章解決「10 個 agent 不夠用」的問題（客製、新開發、接銀行系統）；第 7–8 章上雲與編排；第 9–12 章讓它能在受監理的銀行環境長期活下去。附錄是速查工具。全書內容以本 repo 的 managed-agent-cookbooks/ 與 scripts/ 實際程式碼為準。

  • 02核心概念與 CMA 架構agent.yaml 解剖、callable_agents、安全分層
  • 03開發環境與快速上手mock-mcp 離線跑通、dry-run、開發循環
Part II — 開發：讓 Agent 符合你的業務
  • 04客製化既有 Agent改 prompt / skill / 模型的正確姿勢
  • 05從零開發一個新 Agent授信覆核 Agent 完整 walkthrough
  • 06整合銀行內部系統（MCP）為核心帳務、徵信系統寫 MCP server
Part III — 上雲與串接
  • 07部署：CMA 與 AWS 架構deploy 腳本解析、VPC/Secrets/CI-CD 參考架構

  • 09安全與金融合規最小權限、PII、prompt injection、法遵對映
  • 10治理與模型風險管理變更管理、模型升級、agent 台帳、HITL
  • 11維運、監控與成本可觀測性、runbook、回滾、token 成本
