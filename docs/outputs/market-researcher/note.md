# 🔭 市場研究（market-researcher）— 筆記

## 為什麼是它，而不是別支

- ✅ 唯一一支「沒接任何資料源也跑得動」——核心（產業概覽/競爭格局/點子）靠 Claude 內建網路搜尋就能生出一份結構化草稿
- ✅ 風險最低（🟢）——產出只是內部研究草稿，不對外、不過帳、不簽核，試錯零代價
- ✅ 不需要自建任何 `.mcp.json`——capiq/factset 是公開連接器，缺金鑰只會少了精準的 comps 數字，不會讓 agent 跑不起來
- ✅ 立刻看得到價值——丟一個你不熟的產業/主題進去，就有東西出來

## 實際怎麼起步（三步）

1. 安裝外掛：把 financial-analysis（公開連接器來源） + market-researcher 兩個外掛都裝起來
2. 直接用：丟一個主題給它，例如「研究 AI 資料中心散熱這個主題」 → 此時沒金鑰它會用網路搜尋，先給你一版草稿
3. 想升級精準度再補：之後再去向 FactSet/CapIQ 買訂閱、填 API 金鑰，comps 數字就會從「網路估」變「官方資料」

## 一句話

先用 🔭 市場研究把流程跑順、建立信任，接著往 🧮 估值建模 → 📊 財報追蹤 延伸（這批都不必碰內部系統）；最後再啃要接內部系統的後台與開戶審查。這跟你 AgentSummary 裡的三批導入順序完全一致——你正好可以拿第一批的第一支當試水溫。
